大課題（情報実験第一）

問題

巡回セールスマン問題（じゅんかいセールスマンもんだい、英: traveling salesman problem、TSP）は、都市の集合と各2都市間の移動コスト（たとえば距離）が与えられたとき、全ての都市をちょうど一度ずつ巡り出発地に戻る巡回路の総移動コストが最小のものを求める（セールスマンが所定の複数の都市を1回だけ巡回する場合の最短経路を求める）組合せ最適化問題である。

入力

1行目に正の整数n(<30,000)が与えられる。更に、i(0<i<n)行目には頂点番号(i-1)の頂点の座標であるx_(i-1), y_(i-1)がスペース区切りで与えられる。

出力

全ての頂点をちょうど一度ずつ巡り、出発地に戻る総移動コストがなるべく小さくなるような頂点の訪問順を求め、スペース区切りで頂点番号を1行に出力せよ。ただし、始点/終点となる頂点は、最後にもう一度出力してはならない。また、2行目以降の出力は無視するので好きなようにコメントを書いても良い。

サンプル

入力：

3
0 0
1 0
0 1

出力：

0 2 1
2行目以降は何を書いても良い

採点基準

このサイトでのスコアは、成績評価に含まれないので注意すること。
